# Kapital Bank Testing Project
## Layihe haqqinda
Bu repozitoriya Kapital Bank sistemi uzre hazırlanmış test sənədlərini əhatə edir. Layihə real bank sisteminisimulyasiya edən web və mobil banking tətbiqi ucun hazirlanmısdir.Repozitoriyada  test strategiyası , test plan və sistem arxitekturası və test dioqramları, test scope və risk analizi, entry və exit kriteriyaları mövcuddur .
## Məqsəd
Bu layihənin məqsədi :
- Test sənədlərinin necə hazirlandığını göstərməkə 
- Web və mobil bankçılıq sitemlərinin test edilmə prosesini izah etməkdir
