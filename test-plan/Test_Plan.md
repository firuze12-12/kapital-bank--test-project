# Test Plan - Kapital Bank

## Test Introduction
Bu test plan Kapital Bank rəqəmsal bankçılıq sitemi üçün konkret test fəliyyətlərini , test cədvəlini və resursları təsvir edir . Aşağıdakı elementləri əhatə edir :
- Login funksiyası ( Web və mobil )
- Balance yoxlanması
- Money transfer

  ## Test objectives
  - Əsas bank funksiyalarının düzgün işlədiyini yoxlamaq
  - Kritik biznes axınlarında xətaların qarşısını almaq
  - Web və mobil platfformalarda uyğunluğu təmin etmək

    ## Test İtems
  Aşağıdakı modullar test ediləcək
  - Login module
  - Balance Module
  - Money Transfer Module
 
   ## Test environment
    - Web browser ( chrome / Firefox )
    - Mobil app ( Android / İOS )
    - Test database : QA database
    - Network : Test Network

  ## Test Desing Techniques
  Test case - lər aşağıdakı texnikalara əsaslanacaq :
  - Equivalece Partitioning
  - Boundary Value Analysis
  - Use Case Testing
  - Error Guessing

## Test Scenarios
Login - Positive Scenario
- Düzgün username və password ilə giriş
Login - Negative Scenario
- Yanlış password
- Boş sahələr
- Bloklanmış istifadəçi
  
## Test Execution
Testlər manual şəkildə icra ediləcək və nəticələr test case sənədlərində qeyd olunacaq

  ## Test Deliverables
  - Test Plan
  - Test Strategy
  - Test Cases
  - Bug Reports
  - Test Summary Report

    ## Test Summary
    Test prosesinin sonunda test nəticələri ümumiləşdiriləcək və sistemin keyfiyyəti haqqında qısa hesabat hazırlanacaq
